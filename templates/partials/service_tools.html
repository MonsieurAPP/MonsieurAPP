Script Tampermonkey
Apri l'installazione direttamente nel browser corrente oppure scarica il file come fallback. Lo userscript viene generato per questa istanza della app.
Installa o aggiorna in questo browser Scarica script
Se Tampermonkey e' gia' installato in questo browser, il primo bottone apre la proposta di installazione o aggiornamento dello script in una nuova scheda.
Come si installa
1. Installa Tampermonkey nel browser che usi gia' per Monsieur Cuisine.
2. Premi Installa o aggiorna in questo browser: se Tampermonkey e' attivo, vedrai direttamente la schermata di installazione o aggiornamento dello userscript gia' configurato per il dominio corrente.
3. Se il prompt non compare, usa Scarica script: il browser proporra' il nome monsieur-cuisine-bridge.txt. Poi apri il file e incollane il contenuto in un nuovo script Tampermonkey, oppure importalo manualmente se il browser/estensione lo consente.
4. Lascia attiva l'estensione: quando apri Monsieur Cuisine comparira' il pulsante flottante di MonsieurAPP.
5. Dopo ogni aggiornamento puoi ripetere lo stesso bottone di installazione oppure riscaricare il file e reinstallarlo sopra la versione precedente.

Script Tampermonkey
Scarica la versione aggiornata dello userscript da installare nel browser che usi per Monsieur Cuisine.
Scarica script
Come si installa
1. Installa Tampermonkey nel browser che usi gia' per Monsieur Cuisine.
2. Premi Scarica script da questa pagina e salva il file monsieur-cuisine-bridge.user.js.
3. Apri Tampermonkey e importa il file scaricato, oppure crea un nuovo script e incolla il contenuto del file.
4. Salva lo script e lascia attiva l'estensione: quando apri Monsieur Cuisine comparira' il pulsante flottante di MonsieurAPP.
5. Dopo ogni aggiornamento dello script puoi riscaricarlo da qui e reinstallarlo sopra la versione precedente.
